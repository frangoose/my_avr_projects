- avr_coffee_roaster
  - Hacking a FreshRoast SR500 coffee roaster to enable more functionality
- avr_coffee_timer
  - A timer for brewing coffee
- avr_servo
  - Servo control library
- avr_snake_5110
  - Snake game on Nokia 5110 LCD
- avr_template
  - Blank template for starting projects
